## 技術的知見・学び

### Claude Code ネイティブインストール

| 項目 | 内容 |
|------|------|
| インストールコマンド | `curl -fsSL https://claude.ai/install.sh \| bash` |
| npmインストール | 非推奨（deprecated） |
| 特徴 | Node.js不要、自動アップデート |
| WSL 2 | サンドボックス機能対応 |
| WSL 1 | サンドボックス非対応 |

### Windows → WSL 移行時の注意点

| 項目 | 対応 |
|------|------|
| ファイルパス | `C:/Users/...` → `/home/...` |
| 改行コード | CRLF → LF（`dos2unix`） |
| 実行権限 | `chmod +x` |
| 認証情報 | 再認証必要（`.credentials.json`移行不要） |

### Claude Code 設定ファイル構成

```
~/.claude/
├── settings.json          # グローバル設定
├── statusline.js          # カスタムステータスライン
├── .credentials.json      # 認証情報（移行不要）
├── plugins/
│   ├── config.json
│   ├── installed_plugins.json
│   ├── known_marketplaces.json
│   ├── marketplaces/      # プラグインソース
│   │   ├── anthropic-agent-skills/
│   │   └── claude-plugins-official/
│   └── cache/             # プラグインキャッシュ
│       ├── anthropic-agent-skills/
│       └── claude-plugins-official/
└── projects/              # プロジェクト固有設定
```
