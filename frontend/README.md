# frontend/

フロントエンドアプリケーションを格納するディレクトリです。

## 構成

```
frontend/
└── angular-app/    # Angularアプリケーション
```

## 位置づけ

このプロジェクトの主目的はNestJS + BFFの学習であり、フロントエンドは**最小限の実装**に留めます。

## 学習フェーズ

**Phase 4** で実装します。

## スコープ

### 含まれるもの
- HttpClientでBFFを叩いて動作確認
- 基本的なログイン/ログアウト
- タスク一覧・作成の簡易UI

### スコープ外
- Angularのユニットテスト
- Playwright / Cypress（フロントエンドE2E）
- 複雑なUI/UX
